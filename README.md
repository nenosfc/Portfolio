## 🌟 Acerca del Proyecto

Este es el código fuente de mi **Portfolio Personal**, un proyecto diseñado para mostrar mis habilidades, experiencia y proyectos como **Desarrollador de Aplicaciones Web**.

Hecho con un enfoque en el diseño **responsivo**, la **accesibilidad** y la **optimización** para una experiencia de usuario fluida.

---

## 🛠️ Tecnologías Utilizadas

El portfolio ha sido construido utilizando las siguientes tecnologías clave:

* **HTML5:** Estructura semántica del contenido.
* **CSS3 (Custom y Bootstrap 5):**
    * **Bootstrap 5:** Usado para el diseño **responsivo** y el sistema de *grid* (columnas), asegurando que se vea bien en cualquier dispositivo.
    * **CSS Custom:** Para estilos personalizados y la paleta de colores.
* **JavaScript:** Interactividad y funcionalidad (como la barra de navegación y el *toggle* del CV).

---

## 🏗️ Estructura y Características

El proyecto incluye las siguientes secciones principales:

* **Inicio (Hero):** Presentación personal y enlaces a redes sociales.
* **Sobre Mí & Video CV:** Breve descripción de mi perfil y un CV en formato video.
* **Proyectos:** Muestra de algunos de mis trabajos más relevantes con enlaces a código y demos.
* **CV:** Opción para ver o descargar mi Curriculum Vitae.
* **Contacto:** Formulario de contacto directo para consultas profesionales.
* **Barra de Navegación Fija:** Navegación sencilla y accesible en todo momento.
